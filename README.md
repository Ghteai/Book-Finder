# Book Finder App 

A simple React + Vite app that lets users search for books using the Open Library API.  
Users can search any book and view details.

## Tech Stack
- React
- Vite
- Chakra UI
- Axios
- React Router
- Docker 

## Run Locally

docker build -t book-finder-app .
docker run -p 3000:3000 book-finder-app
